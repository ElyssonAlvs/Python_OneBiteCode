# Minicurso Python OneBiteCode

### Durante o minicurso foi apresentada noções básicas de python para a manipulação de dados.

```markdown
    Importando os Dados
    - Análise do Estudo de Caso

    - Configuração Python

    - Selecionando Colunas de um Dataframe
```
